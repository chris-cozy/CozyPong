# simple pong

<h2>Description</h2>
This is a simple pong clone implemented using Python's Pygame library. It is constructed as a hybrid of an individual game and a game engine.
<br>
<h2>Installation</h2>
To run this project use the terminal command:
<br>
<i>python ***.py</i>
